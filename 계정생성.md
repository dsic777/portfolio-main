# 데모 계정 생성 / 비밀번호 초기화

> EC2 어디서든 실행 가능. 해당 섹션만 복사해서 붙여넣기.
> 모든 계정: **아이디 `test` / 비밀번호 `1234`**

---

## HumanRM — 관리자 계정 (Django)

```bash
docker exec humanrm python manage.py shell -c "
from django.contrib.auth import get_user_model
User = get_user_model()
if not User.objects.filter(username='test').exists():
    User.objects.create_superuser(username='test', password='1234', email='')
    print('test 계정 생성 완료')
else:
    u = User.objects.get(username='test')
    u.set_password('1234')
    u.save()
    print('test 계정 비밀번호 재설정 완료')
"
```

## 말벗2 — 모바일 사용자 계정 (FastAPI + SQLAlchemy)

- 닉네임: `다시날고싶다`

```bash
docker exec malbeot-app python -c "
import asyncio, bcrypt, sys
sys.path.insert(0, '/app')

async def create_user():
    from app.models.user import User
    from app.database import engine
    from sqlalchemy.ext.asyncio import AsyncSession
    from sqlalchemy.orm import sessionmaker
    from sqlalchemy import select

    async_session = sessionmaker(engine, class_=AsyncSession, expire_on_commit=False)
    async with async_session() as session:
        result = await session.execute(select(User).where(User.username == 'test'))
        user = result.scalar_one_or_none()
        pw_hash = bcrypt.hashpw(b'1234', bcrypt.gensalt()).decode()
        if user:
            user.password_hash = pw_hash
            user.nickname = '다시날고싶다'
            print('test 계정 업데이트 완료')
        else:
            session.add(User(username='test', password_hash=pw_hash, nickname='다시날고싶다'))
            print('test 계정 생성 완료')
        await session.commit()

asyncio.run(create_user())
"
```
